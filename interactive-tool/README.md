# Interactive Choking Response Demonstration Tool

This web-based tool demonstrates proper techniques for responding to choking incidents in children of different ages.

## Features

- **Age-specific demonstrations**: Infants, toddlers, and older children
- **Step-by-step instructions**: Visual guides for back blows and abdominal thrusts
- **Interactive practice**: Simulated scenarios with feedback
- **Accessibility**: Designed for screen readers and keyboard navigation
- **Multilingual support**: Language toggle (coming soon)

## Technology Stack

- **HTML5**: Semantic structure
- **CSS3**: Responsive design and animations
- **JavaScript**: Interactive functionality
- **No external dependencies**: Works offline after loading

## Usage

### Online Access
Visit the deployed version at [GitHub Pages link] (when configured)

### Local Development
1. Clone the repository
2. Navigate to `interactive-tool/` directory
3. Open `index.html` in a web browser
4. No build process required

### For Integration
- Embed as an iframe in educational websites
- Use as a training module in healthcare education platforms
- Adapt for mobile applications
