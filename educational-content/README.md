# Educational Content

This directory contains evidence-based educational materials about pediatric choking prevention and response.

## Contents

- **[Common Choking Hazards](common-hazards.md)**: Age-specific lists of high-risk foods, toys, and household items
- **[Prevention Strategies](prevention-strategies.md)**: Guidelines for homes, schools, childcare facilities, and restaurants
- **[Emergency Response](emergency-response.md)**: Step-by-step instructions for choking first aid (back blows, abdominal thrusts, CPR)
- **[Age-Specific Guides](age-specific-guides/)**: Detailed recommendations for different developmental stages
- **[Visual Aids](visual-aids/)**: Diagrams, infographics, and illustrations

## Purpose

The goal of this content is to provide clear, actionable information that can be easily understood by parents, caregivers, educators, and healthcare professionals. All content is reviewed for medical accuracy and regularly updated based on the latest evidence.

## Contributing

To contribute new content or suggest improvements:

1. Review the existing content to avoid duplication
2. Ensure all medical information is backed by reputable sources
3. Use plain language and avoid unnecessary jargon
